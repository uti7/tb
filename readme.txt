
ttba/btta

概要

ttba と btta は開発者向けの構造体バイナリ／テキスト変換フィルタです。
ttba は、規定の書式で記述されたテキストをバイナリデータに変換し、ファイルまたは
標準出力へ出力します。
btta は、任意のデータを ttba 形式のテキストに変換して標準出力へ出力します。
btta には出力フォーマット（変換書式、データ構造など）を与えることができ、それに
従った出力が可能です。

仕事で使いながら少しずつ機能を増やしてきたものです。

別に難しいことをしているわけでもなく、誰でもすぐ考えつくようなツールですが、他
であまり見られないような機能があります。

動作環境

あたらしめの gcc でコンパイルすれば、ほとんどのUNIX系OSで動くと思います (gcc3.x
は未確認)。
以下の環境で動作を確認しています。

    Solaris 2.5.x (gcc 2.95.1)
    Solaris 8 (gcc 2.95.2)
    HP-UX 11.00 (gcc 2.95.2)
    AIX 4 (gcc 2.95.1)
    Windows9x (Cygwin 1.x)
    Windows2OOO (Cygwin 1.x)
    Linux 2.0.x (gcc 2.95.2)

[2003/03/09] Windows Me/2OOO (VC++6.0)ポート版を置きました（このページの一番下)
